# <a name="faqs"></a>FAQ

Voici le Forum aux questions ayant des utilisateurs et organisations sur administration et l’utilisation de Microsoft Search (recherche Microsoft).
  
## <a name="whats-the-value-proposition-for-microsoft-search"></a>Quelle est la proposition de valeur pour Microsoft Search (recherche Microsoft) ?

Microsoft Search (recherche Microsoft) est un moyen sécurisé pour rechercher votre travail et votre contenu web. Ce type d’intégration web et entreprise est disponible uniquement avec Microsoft. Pour plus d’informations liées à la conformité, voir [sécurité relative à Microsoft Search (recherche Microsoft)](security.md).
  
## <a name="what-microsoft-search-features-are-available-now"></a>Quelles sont les fonctionnalités de Microsoft Search désormais disponibles ?

Pour une liste complète, voir [fonctionnalités de Microsoft Search (recherche Microsoft)](features.md).
  
## <a name="does-microsoft-search-support-advanced-query-understanding"></a>Est-ce que Microsoft Search(recherche Microsoft) prend en charge une requête de présentation avancée ?

Oui, Microsoft Search (recherche Microsoft) analyse objectif requête à partir de phrases plus grande. Cette fonctionnalité utilise AI pour en savoir plus sur les phrases courantes superflues que les utilisateurs ajoutent à leurs requêtes qui n’ont pas d’impact sur leur objectif de recherche. Par exemple, lorsqu’un utilisateur recherche « en savoir plus sur comment modifier mon mot de passe svp» nous extrayons les mots moins importants à partir de la requête, puis déclenchons ceux pertinents par exemple, « modifier votre mot de passe ».
  
Cette fonctionnalité ne remplace pas l’ensemble de mots clés dans le portail d’Administration.
  
## <a name="does-microsoft-search-search-for-files-on-premises-as-well-as-the-cloud"></a>Microsoft Search (recherche Microsoft) recherche des fichiers en local, ainsi que le cloud ?

Microsoft Search (recherche Microsoft) prend en charge la recherche pour les documents live sur SharePoint Online, OneDrive entreprise, ainsi que sur hybride en local SharePoint et SharePoint Online pour les types de fichiers Office courants. Si vous pouvez rechercher du contenu local dans SharePoint Online, vous devriez effectuer une recherche de contenu local à l’aide de Microsoft Search (recherche Microsoft). 
  
## <a name="does-microsoft-search-replace-other-enterprise-search-experiences"></a>Microsoft Search (recherche Microsoft) remplace autres expériences de recherche d’entreprise ?

Microsoft Search (recherche Microsoft) est une offre complémentaire qui regroupe les résultats de plusieurs sources. Pour plus d’informations sur les sources et types de fichiers pris en charge, voir le Forum aux questions suivant.
  
## <a name="what-file-types-and-sources-does-microsoft-search-support"></a>Quels types de fichiers et de sources sont pris en charge par Microsoft Search (recherche Microsoft)?

Nous prenons en charge les sources de contenu suivants :
  
- SharePoint Online
    
- SharePoint hybride (local + SPO)
    
- OneDrive Entreprise
    
Les types de fichiers suivants favorisés par recherche de fichiers et s’affichent sous l’onglet fichiers pour les personnes et groupes :
  
- Word
    
- Excel
    
- PowerPoint
    
- OneNote
    
- PDF
