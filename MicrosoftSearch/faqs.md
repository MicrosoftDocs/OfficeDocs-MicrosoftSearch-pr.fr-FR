# <a name="faqs"></a>FAQ

Voici les questions fréquemment posées organisations et les utilisateurs ont sur l’administration et à l’aide de Microsoft Search.
  
## <a name="whats-the-value-proposition-for-microsoft-search"></a>Quelle est la proposition de valeur pour Microsoft Search ?

Microsoft Search est un moyen sécurisé pour rechercher votre travail et le contenu web. Ce type de l’intégration entre web et d’entreprise est disponible uniquement avec Microsoft. Pour plus d’informations relatives à la conformité, voir [Security for Microsoft Search](security.md).
  
## <a name="what-microsoft-search-features-are-available-now"></a>Les fonctionnalités de Microsoft Search sont maintenant disponibles ?

Pour obtenir la liste complète, voir [Fonctionnalités de Microsoft Search](features.md).
  
## <a name="does-microsoft-search-support-advanced-query-understanding"></a>Est Microsoft Search prise en charge avancée requête présentation ?

Oui, Microsoft Search analyse l’objectif de requête à partir d’expressions plus grandes. Cette fonctionnalité utilise AS pour en savoir plus courantes utilisateurs expressions superflus ajoutent à leurs requêtes ne pas avoir un impact sur leur mode de recherche. Par exemple, lorsqu’un utilisateur recherche « en savoir plus sur comment changer mon mot de passe, » nous extraire les mots moins importantes à partir de la requête et le déclencheur selon celles pertinents tels que « modifier le mot de passe ».
  
Cette fonctionnalité ne remplace pas l’ensemble de mots clés dans le portail d’administration.
  
## <a name="does-microsoft-search-search-for-files-on-premises-as-well-as-the-cloud"></a>Microsoft Search recherche pour les fichiers locaux, ainsi que le nuage ?

Microsoft Search prend en charge la recherche de documents sur SharePoint Online, OneDrive entreprise, ainsi que sur hybride local SharePoint et SharePoint Online pour les types de fichiers Office les plus courants. Si vous pouvez rechercher le contenu local dans SharePoint Online, vous devez pouvoir rechercher du contenu sur site à l’aide de Microsoft Search. 
  
## <a name="does-microsoft-search-replace-other-enterprise-search-experiences"></a>Microsoft Search remplace d’autres expériences de recherche de contenu d’entreprise ?

Microsoft Search est une offre complémentaire qui rassemble les résultats à partir de plusieurs sources. Pour plus d’informations sur les sources et les types de fichiers pris en charge, voir le Forum aux questions suivant.
  
## <a name="what-file-types-and-sources-does-microsoft-search-support"></a>Quels types de fichiers et les sources de prise en charge par Microsoft Search ?

Nous prennent en charge les sources de contenu suivants :
  
- SharePoint Online
    
- SharePoint hybride (local + SPO)
    
- OneDrive Entreprise
    
Types de fichiers suivants sont exposées dans la recherche de fichiers et apparaissent sous l’onglet fichiers pour les personnes et groupes :
  
- Word
    
- Excel
    
- PowerPoint
    
- OneNote
    
- PDF
